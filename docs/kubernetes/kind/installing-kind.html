Installing the kind CLI

Procedure​

  • In the status bar, click on Kind, and follow the prompts.

Verification​

  1. The status bar doesn't display Kind.

  2. Settings > Resources contain a Kind tile.

  3. You can run the kind CLI:

    $ kind get clusters
